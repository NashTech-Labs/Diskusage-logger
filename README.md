# Disk Usage Logger

## Prerequisites

- Ansile
- Linux Machine

## Setup

Run the bash script and relax

```bash
git clone https://github.com/Rahul-Soni28/disklogger.git
cd disklogger
bash setup.sh <yoursudopassword>
```

## Check logs

Logs will be generated at 12:00 pm everyday

```bash
cat /var/log/diskusage.log
```
